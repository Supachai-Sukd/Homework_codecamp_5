1.ให้หาว่า Attribute ของ Tag ต่อไปนี้มีอะไรบ้างและใช้งานอย่างไร

ชื่อ Tag ลักษะณะการใช้งาน
H ใช้ประกาศหัวข้อของเนื้อหาในส่วนนั้นๆ
p paragraph นั้นใช้ในการอธิบายสิ่งต่างๆโดยทุก tag p จะขึ้นย่อหน้าใหม่ทุกครั้ง
div division คือ tag ที่ช่วยในการจัดการหน้าเว็ปให้เป็นส่วนๆเป็น group เช่น div
นี้เราต้องการให้มี class อะไรห่างตรงนั้นเท่าไรอีก div นึงต้องการตัวหนังสือคนละสีกับอีก div นึงเป็นต้น
img ใช้ในการ ใส่รูปภาพ
a ใช้ในการทำ Link ต่างๆไม่ว่าจะเป็น text link,button link เป็นต้น
ul unorder list ใช้แสดงรายการเป็น List โดยข้างหน้าเป็น bullet จุดดำๆไม่มีตัวเลข
ol order list ใช้แสดงรายการเป็น list โดยข้างหน้าเป็นตัวเลขกำกับ
table ใช้ในการสร้างตารางส่วนใหญ่ใช้ในการแสดงข้อมูลในหน้าเว็ป
form ใช้ทำเป็นฟอร์มในการลงทะเบียน ล๊อกอิน หรือ อื่นๆ
button เป็นปุ่มใช้ในการ submit cancel form หรือทำเป็น link ได้เป็นต้น

2.ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

Tag ความเหมือน ความต่าง
hr & br เป็นการเว้นระยะของแต่ละส่วนที่เราต้องการ ต่างกันตรง hr เป็นขีดยาว br ไม่มีขีด
ul & ol แสดงรายการเป็น list เหมือนกัน ul เป็น bullet จุดดำๆข้างหน้า ol เป็นตัวเลข
select & checkbox & radio button เป็นการเลือกเหมือนกัน เช่น เลือกเพศ select เป็นแบบ drop down
checkbox เป็นแบบช่องติ๊กถูกเลือกได้มากกว่า 1
radio button เป็นช่องกลมๆให้เลือกและเลือกได้อันเดียว
div & span เป็นการแบ่งกลุ่มของ code ช่วยในการจัดการหน้าเว็ปให้ง่าย div เป็นการแบ่งกลุ่ม code เป็นแนวตั้งเช่น div อันที่ 2 ต่อจาก div บนลงมาข้างล่างแต่การใช้งานในระดับสูงสามารถประยุกต์ใส่ class ต่างๆให้อยู่แนวนอนหรือตั้งได้หมด
span เป็นการแบ่งกลุ่มออกไปทางแนวนอน
script & style เป็น tag ที่ใช้สำหรับใส่ code ที่ไม่ใช่ html เหมือนกัน script เป็น tag ที่ใช้สำหรับใส่ code javascript เข้าไปข้างใน
style เป็น tag สำหรับใส่ code css เข้าไปข้างใน

3.ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร

ประเภท Input หน้าที่
input type="text" Input ประเภทข้อความ
input type="password" Input ประเภทรหัสผ่านตอนกรอกจะเป็นกลมๆไม่แสดงตัวหนังสือ
input type="submit" Input นิยมใช้กับ form กดแล้วจะส่งข้อมูลใน form ที่กรอกไว้ไปยัง server
input type="reset" Input นิยมใช้กับ form กดแล้วจะ clear form ไปค่า default
input type="radio" Input นิยมใช้กับ form เป็นตัวเลือกให้ user เลือกและสามารถเลือกได้แค่อันเดียว
input type="checkbox" Input นิยมใช้กับ form เป็นตัวเลือกให้ user สามารถเลือกกี่อันก็ได้หรือไม่เลือกก็ได้
input type="button" เป็นการสร้างปุ่มอยากให้ทำอะไร้เพิ่มต้องระบุ value เพิ่ม
input type="color" Input ใช้สำหรับเลือกสี
input type="date" Input ใช้สำหรับเลือกวันที่
input type="datetime-local" Input ใช้สำหรับเลือกวันที่แบบไม่อ้างอิง time zone
input type="email" Input ใช้สำหรับกรอก email
input type="file" Input ใช้สำหรับเลือก file ที่จะ upload
input type="month" Input ใช้สำหรับเลือกเดือน
input type="number" Input ใช้สำหรับกรอกตัวเลขเท่านั้น
input type="range" Input ใช้สำหรับเลือกปริมาณที่เกี่ยวกับตัวเลขเป็นแบบ bar เลื่อนไปมา เช่น เพิ่ม ลด เสียง
input type="search" Input ใช้สำหรับค้นหาข้อมูล
input type="tel" Input ใช้สำหรับใส่ข้อมูลเบอร์โทรศัพท์
input type="time" Input ใช้สำหรับเลือกเวลา
input type="url" Input ใช้สำหรับใส่ link url เช่น เว็ปของตัวเองเป็นต้น
input type="week" Input ใช้สำหรับเลือกสัปดาห์
