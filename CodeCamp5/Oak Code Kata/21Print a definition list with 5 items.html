Kubernetes
” Kubernetes ” หรือ “K8s” คือ “Container Orchestration”เป็น OpenSource จาก Google ที่จะมาช่วย ในการ จัดการคอนเทนเนอร์ ( เช่น Docker, LXC)
React js
React เป็น JavaScript Library ที่ถูกสร้างโดย Facebook
Node js
Node.js คือ Cross Platform Runtime Environment สำหรับฝั่ง Server และเป็น Open Source ซึ่งเขียนด้วยภาษา JavaScript
Express js
Express เป็น web application framework บน Node.js ที่ได้รับความนิยมมากๆตัวหนึ่งครับ ซึ่งตัว Express เนี่ยจะมีฟีจเจอร์ต่างๆที่ช่วยให้เราทำเว็บได้สะดวกขึ้น เช่น การทำ routing, middleware การจัดการ request และ response เป็นต้น
Mongoose
Mongoose เป็น ODM (Object Document Mapping) หน้าที่ของมันก็เหมือนกับ ORM ฝั่ง DBMS
