UX
	1.Interaction Design
		1.1 มีการ Search สินค้าด้วยการพิมพ์ทาง Keyboard
		1.2 มีการใส่เบอร์โทรของผู้ขายเพื่อให้มีการตอบโต้คุยรายละเอียดกันด้วยเสียง
        2. Information Architect
		2.1 มีการโชว์ประเภทสินค้ายอดฮิตไว้บนสุดเพื่อให้ลูกค้า่หาข้อมูลได้ง่าย
		2.2 เลื่อนลงมาเรื่อยๆจะมีประกาศสินค้าใหม่ๆแต่ละหมวดไล่ไปเรื่อยๆ
        3. Scenarios
		3.1 User อาจจะเข้ามาแค่สำรวจราคาเทียบกับแหล่งอื่นๆ
		3.2 User อาจจะเข้ามาหาสินค้าราคาถูกไปขายต่อ
		3.3 User อาจจะเข้ามาดูอะไรเพลินๆเจออะไรถูกใจค่อยซื้อ
		3.4 User จงใจเข้ามาหาสินค้าที่ตัวเองต้องการ
UI
	1. Color
		1.1 ใช้สีม่วงเป็นโทนหลักดูสบายตาส่วนสำหรับผูู้ขายก็มีการเน้นปุ่ม
ขายให้เป็นสีส้มเพื่อความโดดเด่น
        2. Graphic Design
		2.1 การออกแบบ Logo เป็นแค่ตัวหนังสือแต่ว่ามีเอกลักษณ์เห็นแล้วสามารถจำได้ง่าย
                2.2 Icon ต่างๆดูดีง่ายๆเป็นเอกลักษณ์
        3. Layout
		3.1 การจัดวางในหน้าแรกถือว่าออกแบบได้ดีมีการเอาหมวดหมู่ยอดฮิตไว้บนสุดจัดวางโดยใช้ Card ทำให้ดูเรียบง่ายสบายตา
		3.2 การจัดวาง Navbar สามารถใช้พื้นที่ได้คุ้มโดยการให้ความสำคัญกับช่อง Search 
        4. Font
		4.1 รูปแบบ Font ใช้ Font ที่ดูแล้วไม่หงุดหงิด
		4.2 สี Font มีการแบ่งประเภทของ Font ถ้า Font คนละประเภทก็คนละสีชัดเจน เช่น หัวข้อสีนึง เนื้อหาสีนึง
