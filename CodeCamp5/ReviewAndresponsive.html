    Class selector

    ในการกำหนด selector นั้นเราจะใช้ . หรือ จุดในการกำหนด selector ประเภท class ตามรูปตัวอย่างข้างบน แล้วเราต้องการให้มีคุณสมบัติอะไรเราก็ใส่ไปในการกำหนด selector ของอันนั้นๆส่วนวิธีการใช้นั้นใน tag html ที่เราต้องการใช้นั้นเราก็พิมพ์คำว่า class ตามรูปตัวอย่าง การใช้ selector แบบ class นั้นมีข้อดีคือ จะยืดหยุ่นกว่าแบบ ID เพราะ ใน 1 html tag ที่มีการใช้ class ต่างๆเราสามารถใส่ได้มากกว่า 1 class
