    Class selector

    ในการกำหนด selector นั้นเราจะใช้ . หรือ จุดในการกำหนด selector ประเภท class ตามรูปตัวอย่างข้างบน แล้วเราต้องการให้มีคุณสมบัติอะไรเราก็ใส่ไปในการกำหนด selector ของอันนั้นๆส่วนวิธีการใช้นั้นใน tag html ที่เราต้องการใช้นั้นเราก็พิมพ์คำว่า class ตามรูปตัวอย่าง การใช้ selector แบบ class นั้นมีข้อดีคือ จะยืดหยุ่นกว่าแบบ ID เพราะ ใน 1 html tag ที่มีการใช้ class ต่างๆเราสามารถใส่ได้มากกว่า 1 class

    ID Selector

    ตัวอย่างการใช้งาน ID selector

    ตัวอย่าง id selector ใน P tag

    เป็นการกำหนด selector อีกแบบหนึ่ง ซึ่งเราจะใช้สัญลักษณ์ # นำหน้า selector ของเรา เราควรเลือกใช้ ID เมื่อเราต้องการระบุถึงส่วนสำคัญของหน้าที่ในหน้าหนึ่งๆจะมีแค่หนึ่งเดียวเท่านั้น เช่น ส่วนหัวด้านบนของหน้า, ส่วนเมนูหลักของหน้า, ส่วนเนื้อหาของหน้า หรือส่วนท้ายสุดที่ประกอบในหน้า หรือเลือกใช้ ID ในส่วนที่เราต้องการใส่

    Descendant combinators

    คือ การเขียน Selector เพื่อสั่งการทำงานทั้งหมด ทั้ง ลูก หลาน เหลน โหลน โดยการเขียนนั้น ใช้ "ช่องว่าง" ในการสั่งงาน ตัวอย่างตามรูปด้านล่าง ความหมายคือ division ที่มี class show ให้ tag paragraph ใน division นี้ใช้สี crimson

    ข้างบนที่ใส่ class เผื่อไปทำตัวอย่างอื่นจะได้ไม่ซ้ำกัน ส่วน tag นี้อยู่ใน division เดียวกันกับ tag p แต่ว่าไม่โดนใช้สีเดียวกันกับ tag p เพราะไม่ตรงกับเงือนไข CSS ที่เราประกาศนั่นเองตัวอย่างรูปด้านล่าง

    Child combinators

    คือ การใช้งานจะเหมือนๆกับ Descendant ไม่ใช่สั่งงานแค่ลูกต่างกันแค่เราถนัดอะไร รูปตัวอย่างด้านล่าง แต่ถ้าเรา ต้องการอะไรที่เฉพาะเจาะจงเราสามารถใส่ class หรือ id ที่ css ได้เลยและใน html tag เราก็เรียกใช้ class หรือ id นั้นๆ

    Property Value Unit Different
    คือชื่อคุณสมบัติของ css เช่น width, height, position ฯลฯ คือค่าของ property นั้นๆ ครับ เช่น 500px สำหรับความกว้าง เป็นหน่วยต่างๆ เช่น
    1. px เป็นการกำหนดขนาดแบบตายตัว คือกำหนดไว้เท่าไรคือเท่านั้นเลย
    2. em เป็น relative ซึ่งขนาดของ em นั้น จะขึ้นอยู่กับ parent element
      ที่อยู่ใกล้ที่สุดโดย 1 em จะเท่ากับ 1 เท่าของขนาด font-size ของ parent element
    3. rem นั้นก็คล้าย ๆ กับ em คือเป็น relative จะมีจุดต่างกันก็คือ rem
      นั้นหมายถึง (root em) ซึ่งค่าของมันไม่ได้ขึ้นอยู่กับ element parent แต่จะขึ้นอยู่กับขนาดของ root element
    		 ไม่มีอะไรแตกต่างแต่เราต้องเข้าใจว่า css มี property : value และใน value พวกหน่วยที่เกี่ยวกับขนาดต่างๆจะมี unit เป็นการกำหนดค่าของ value นั้นๆเช่น font-size: 40px; เป็นต้น

    Display Value Grid Flex Justify and Align
    1. inline
      • แสดงองค์ประกอบเป็นองค์ประกอบแบบอินไลน์ เช่น span คุณสมบัติความสูงและความกว้างใด ๆ จะไม่มีผลกระทบ
    2. block
      • แสดงองค์ประกอบเป็นองค์ประกอบบล็อก (เช่น p) มันเริ่มต้นในบรรทัดใหม่และรับความกว้างทั้งหมด
    3. contents
      • ทำให้คอนเทนเนอร์หายไปทำให้องค์ประกอบย่อยเป็นองค์ประกอบย่อยขององค์ประกอบระดับถัดไปใน DOM
    4. flex
      • แสดงองค์ประกอบเป็นคอนเทนเนอร์ flex ระดับบล็อก
    5. grid
      • แสดงองค์ประกอบเป็นคอนเทนเนอร์กริดระดับบล็อก
    6. inline-block
      • แสดงองค์ประกอบเป็นคอนเทนเนอร์บล็อกระดับแบบอินไลน์ องค์ประกอบนั้นถูกจัดรูปแบบเป็นองค์ประกอบแบบอินไลน์
        แต่คุณสามารถใช้ค่าความสูงและความกว้างได้
    7. inline-flex
      • แสดงองค์ประกอบเป็นคอนเทนเนอร์แบบยืดหยุ่นในระดับ
    8. inline-grid
      • แสดงองค์ประกอบเป็นคอนเทนเนอร์กริดในระดับ
    9. inline-table
      • องค์ประกอบนั้นจะแสดงเป็นตารางระดับอินไลน์
    10. list-item
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ li
    11. run-in
      • แสดงองค์ประกอบเป็นบล็อกหรืออินไลน์ขึ้นอยู่กับบริบท
    12. table
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ table
    13. table-caption
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ caption
    14. table-column-group
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ colgroup
    15. table-header-group
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ thead
    16. table-footer-group
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ tfoot
    17. table-row-group
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ tbody
    18. table-cell
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ td
    19. table-column
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ col
    20. table-row
      • ปล่อยให้องค์ประกอบทำงานเหมือนองค์ประกอบ tr
    21. none
      • องค์ประกอบจะถูกลบออกอย่างสมบูรณ์
    22. initial
      • ตั้งค่าคุณสมบัตินี้เป็นค่าเริ่มต้น
    23. inherit
      • สืบทอดคุณสมบัตินี้จากองค์ประกอบพาเรนต์
    อันนี้คือการประกาศ CSS แบบ Class โดยการเขียนนั้นบรรทัดแรกคือบอกว่า อันนี้คือ display แบบกริดนะ บรรทัด 2 คือ การประกาศความสูงในแนวตั้งให้สูง 150px แล้ว กำหนดให้มี auto 5 อัน ความหมายคือเป็นการกำหนดคอลั่มนะว่าใน 1 row เนี่ยมี 5 Column ที่เหลือ คือ ช่องว่างระหว่างกัน สี ช่องว่างจากขอบ
    อันนี้จะเหมือน div.row ของ boostrap แต่ boostrap จะมี std 12 column
    บรรทัดแรก สี บรรทัดสอง ข้อความอะไรใน div ให้อยู่กลาง บรรทัดสามคือ ระยะห่างจากขอบ 20 ส่วน 0 คือ เป็นตำแหน่ง Default ของการใช้้ padding ถ้าเราปรับตัวเลขมากขึ้นหรือน้อย ลงตัวหนังสือก็จะขยับขึ้นลงตาม บรรทัดสี่ ขนาดตัวอักษร สรุปความหมายของอันนี้คือ div ที่มี class grid-container แล้ว div ข้างในให้มีคุณสมบัติตามข้างในนี้
